Este programa gera um gif, chamado som.gif, mostrando a evolução do mapa da distância média dos pesos e as coordenadas do neurônio vencedor para cada padrão dos dados de treinamento a cada 1000 iterações do treinamento, a rede foi treinada num total de 100000 iterações, para cada padrão no conjunto de dados, o neurônio vencedor correspondente foi marcado. Cada tipo de marcador representa uma classe de dados das falhas (as classes são falha_1 e falha_2 e são representadas respectivamente com as cores vermelha e verde, respectivamente), o mapa de distância média dos pesos é usado como fundo (os valores são mostrados na barra de cores à direita).
Além disso o programa gera a figura TSNE.png, mostrando os dados de treinamento a partir de duas características principais, esta figura é gerada pelo método TSNE da biblioteca SKLEARN. 
Os dados de treinamento gerando as falhas 1 e 2 do simulador TE, os dados gerados para as falhas podem ser vistos nos arquivos .csv gerados pelo programa.
Para executar esse programa basta escreve python3 TE.py em terminal do sistema Linux.
